Data
Private by design. Everything — history, favorites, keys, cache — lives in this phone's local storage. No analytics, no tracking, no account, no ads, no limits. The only network call this app ever makes is to the translation API you configured above.
Offline — what works without internet
✅ Phrasebook, favorites, history, every cached translation, text-to-speech, currency converter, emergency screen.

⚠️ Needs internet: live speech recognition (Apple processes audio server-side) and new translations. The status pill up top always tells you which mode you're in.

💡 Tip: before going offline, translate the sentences you expect to need — they'll be cached and instant.
Install on iPhone
Open this file in Safari → Share → Add to Home Screen. It launches full-screen like a native app. Mic input requires HTTPS or a Home-Screen install; if the mic button says recognition is unavailable, use Hold-to-talk from a hosted copy or the typed input, phrasebook and cache — they always work.
